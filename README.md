- **[Polymarket Shadow Stack](https://github.com/FiNdAlMkSkInDaL/Polymarket_bot)** — A live L2 order-book data lake and hourly shadow-testing system built around compressed Parquet storage, self-healing metadata and resource-aware evaluation.
- **[Macro Bias](https://github.com/FiNdAlMkSkInDaL/Macro-Bias)** — An automated equity and crypto regime-research platform with quantitative scoring, daily briefings, track records, paper trading and multi-channel distribution.
- **[Latent Control Lab](https://github.com/FiNdAlMkSkInDaL/Zero-Dimensional-API)** — A zero-generation ML experiment that maps frozen DistilGPT-2 hidden states to deterministic VectorBot actions through a confidence-gated probe.
